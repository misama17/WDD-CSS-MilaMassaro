**WEEK 3**

**woensdag 18 feb**

Wat heb ik vandaag gedaan?

Eerst was er een kick-off, waarin er een soort quizje was voor de CSS selectors. Ik vond het erg lastig om de kortste variant hiervan te bedenken. Maar toen ik de voorbeelden zag begreep ik wel wat er stond. Ik moet me wel wat meer verdiepen in de leestekens die je kan gebruiken in CSS.
Daarna werden we ingedeeld in groepjes en ik kreeg het onderwerp Scrolldriven Animations.
We hebben als groepje een thema bedacht en kort besproken hoe we het het beste aan konden pakken.
Ik had de laatste section (over sterren). Ik heb heel veel geëxperimenteerd met verschillende animaties. Ik heb bijvoorbeeld 2 variaties van een zoom-in gemaakt: eentje waar de scale verandert en eentje waar de plek op de z-as verandert met translate. Ik heb dit uiteindelijk niet volledig uitgewerkt voor de presentatie, omdat het nogal een dingetje is met het timen van de keyframes. 
Ik heb uiteindelijk 2 voorbeeldjes uitgewerkt, waarvan ik er eentje zelf had bedacht en de ander van één van de workshops die we hebben gehad en dan iets aangepast, maar min of meer hetzelfde om het te laten zien. 


Hoeveel tijd heeft dit me gekost?

7-8 uur totaal.
90 min kick-off.
rest van de dag (en thuis) nog aan eigen onderdeel gewerkt.


Wat heb ik geleerd?

Dat ik steeds hele ambitieuze ideeën heb, welke voor mij nog moeilijk zijn om uit te werken. Hierdoor wordt ik de hele tijd heel gestresst, omdat ik dan mijn website nog niet af heb. Misschien is het beter om kleiner te beginnen en dan stapje voor stapje groter te maken.
Ik denk heel visueel. Dus ik heb een idee in mijn hoofd wat me heel cool lijkt en dan wil ik dat heel graag uitwerken, maar eigenlijk heb ik geen idee hoe ik dat met code zou kunnen doen. Misschien moet mijn aanpak iets veranderen voor coderen... misschien inspiratie opzoeken van tevoren van echt gecodeerde dingen en niet bijvoorbeeld animatievideo's zoals ik nu had gedaan. Ik denk hetzelfde als dat ik met een Adobe programma bijvoorbeeld zou denken, maar zo ver ben ik nog niet met code en het duurt ook wel even voordat ik op dat punt ben.


Wat ga ik morgen doen?

Morgen heb ik presentatie van ons onderdeel en presentaties van andere groepjes.
Daarna hebben we de kick-off voor de eindopdracht. Ik denk dat ik daarna schetsen ga maken en ga beginnen met een eerste versie.
